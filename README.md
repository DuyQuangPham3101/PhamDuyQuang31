# ThermalStitcher - Ghép Hóa Đơn In Nhiệt

Công cụ web đơn giản giúp ghép hàng loạt ảnh hóa đơn thành một dải ảnh dọc duy nhất, tối ưu cho việc in liên tục trên máy in nhiệt (khổ 58mm hoặc 80mm).

## Tính năng chính
- 🚀 **Chạy trực tiếp trên trình duyệt:** Không cần cài đặt, xử lý ảnh client-side bằng HTML5 Canvas cực kỳ bảo mật (không tải ảnh lên server).
- 📏 **Tùy chỉnh khổ in:** Hỗ trợ khổ 58mm (384px), 80mm (576px) hoặc giữ nguyên độ phân giải gốc.
- ↔️ **Khoảng cách linh hoạt:** Tùy chỉnh khoảng trống giữa các hóa đơn (0px, 8px, 16px, 24px, 32px).
- ✂️ **Đường cắt nét đứt:** Tự động vẽ đường cắt nét đứt giúp dễ dàng cắt giấy sau khi in.
- 🔄 **Sắp xếp thứ tự:** Hỗ trợ kéo thả và đổi thứ tự hóa đơn trước khi ghép.
- 💾 **Tải xuống nhanh chóng:** Xuất ảnh PNG chất lượng cao sẵn sàng để in.
